# google-rpc

This is js compiled version of Google APIs's [RPC protobuf](https://github.com/googleapis/googleapis/tree/master/google/rpc)

## Installation

**Using Yarn**
```bash
yarn add --dev @privyid/google-rpc
```

**Using NPM**

```bash
npm install --save-dev @privyid/google-rpc
```
